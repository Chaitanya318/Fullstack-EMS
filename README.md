# Employee Management System (EMS)

A modern **Full Stack MERN Employee Management System** built with **MongoDB, Express.js, React.js, and Node.js**. The application allows administrators to manage employees, departments, leave requests, payroll information, attendance, and profile settings through a clean and responsive dashboard.

🌐 **Live Demo:** https://fullstack-ems-theta.vercel.app

---

## ✨ Features

### 🔐 Authentication
- JWT Authentication
- Secure Password Hashing with bcrypt
- Protected Routes
- Login & Logout
- Role Based Access (Admin)

### 📊 Dashboard
- Total Employees
- Total Departments
- Today's Attendance
- Pending Leave Requests
- Quick Statistics

### 👨‍💼 Employee Management
- Add New Employee
- Edit Employee Details
- Delete Employee
- Employee Profile
- Department Assignment
- Search Employees
- Filter by Department

### 🏢 Department Management
- Create Departments
- Update Departments
- Department-wise Employee Listing

### 📅 Leave Management
- Apply Leave
- Approve Leave
- Reject Leave
- Leave Status Tracking
- Leave History

### 💰 Payroll
- Employee Salary Records
- Payslip Management
- Payroll Details

### 👤 Profile & Settings
- Update Profile
- Update Password
- Account Settings

### 📱 UI Features
- Responsive Design
- Modern Dashboard
- Clean User Interface
- Fast Navigation
- Mobile Friendly
